# 일상으로 보는 대학생의 문화생활
#### 팀장 : 박연아 팀원 : 김민영 외 32명

## 주제 
#### 대학생들의 생활방식이 문화생활에 어떤 영향이 있는 지 파악한 후 분석 결과를 활용해 기업 입장에서 마케팅 전략을 세움.

## 언어 및 알고리즘
#### 언어 : R, SAS
#### 알고리즘 : Logistic Regression
